# 价格
## 免费计划
暂不支持**免费计划**

## Plus 计划
5 美金/月/每个Workspace ( 工作区 )

- 50人以下
- 日活跃用户30人以下
  - 日活跃用户指的是在一天（24小时）中登录的用户有多少，无论是否使用功能。



## Pro 计划
15 美金/月/每个Workspace ( 工作区 )

- 500 人以下
- 日活跃用户200人以下
  - 日活跃用户指的是在一天（24小时）中登录的用户有多少，无论是否使用功能。

## Team 计划
50 美金/月/每个Workspace ( 工作区 )
- 2000 人以下
- 日活跃用户700人以下
  - 日活跃用户指的是在一天（24小时）中登录的用户有多少，无论是否使用功能。

## 企业计划
暂不支持日活跃人数超过700人或者超过2000人。
